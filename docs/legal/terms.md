# 利用規約

最終更新日: 2026年2月12日

本利用規約(以下、「本規約」)は、本サービス(以下、「本サービス」)の利用条件を定めるものです。ユーザーは本規約に同意の上、本サービスを利用します。

## 第1条(定義)

- 「ユーザー」とは、本サービスを利用するすべての者をいいます。
- 「登録ユーザー」とは、アカウント登録を行ったユーザーをいいます。
- 「コンテンツ」とは、ユーザーがアップロードするPDF、テキスト、画像、入力データ等をいいます。

## 第2条(本サービスの内容)

1. 本サービスは、ユーザーがアップロードしたPDFを解析し、AIを用いて内容に関するチャット応答を提供します。
2. 本サービスは、無料プランおよび有料プランを提供します。
3. 運営者は、サービス内容を予告なく変更、停止または終了できます。

## 第3条(アカウント)

1. ユーザーは、正確な情報を登録しなければなりません。
2. ユーザーは、アカウント情報を自己の責任で管理します。
3. 不正利用が疑われる場合、運営者はアカウントを停止できます。

## 第4条(料金および支払)

1. 有料プランは、表示された月額料金に基づき請求されます。
2. 支払はStripe等の外部決済サービスを通じて行われます。
3. 途中解約の場合も、原則として返金は行いません。
4. 料金およびプラン内容は変更される場合があります。
5. 決済情報は決済事業者により管理され、当社はカード情報を保持しません。

## 第5条(禁止事項)

ユーザーは以下の行為を行ってはなりません。

- 法令に違反する行為
- 著作権侵害となるPDFのアップロード
- 個人情報を不正取得または公開する行為
- システムに過度な負荷をかける行為
- リバースエンジニアリング
- 不正アクセス

違反が確認された場合、アカウント停止や削除を行います。

## 第6条(コンテンツの取扱い)

1. ユーザーは、自らがアップロードするコンテンツについて、必要な権利を有していることを保証します。
2. 本サービスはAI処理のため、コンテンツを一時的に保存する場合があります。
3. 運営者は、サービス改善目的で統計情報を利用できます。
4. 権利者から削除依頼があった場合、当社は当該コンテンツを削除し、必要に応じてアカウントを停止します。

## 第7条(AI応答の性質)

1. 本サービスの回答はAIによる生成結果であり、正確性を保証しません。
2. 医療、法律、投資判断等の専門用途への利用は自己責任とします。
3. 出力結果の正確性や妥当性は、ユーザーが確認するものとします。
4. 生成結果の著作権等の取扱いは、ユーザーが適用法令および関係規約を確認の上、適切に扱うものとします。

## 第8条(免責事項)

本サービスの利用により生じた損害について、運営者は責任を負いません。サービス停止、データ消失、外部API障害、決済障害についても同様です。

## 第9条(知的財産権)

本サービスに関する著作権、商標権等は運営者に帰属します。

## 第10条(過度利用の制限)

通常利用の範囲を超える利用や、システムに高い負荷を与える利用が確認された場合、運営者は利用制限や停止を行うことがあります。

## 第11条(未成年の利用)

未成年のユーザーは、親権者等の同意を得た上で本サービスを利用するものとします。

## 第12条(規約変更)

本規約は、予告なく変更される場合があります。

## 第13条(準拠法)

本規約は日本法に準拠します。
