# Legal Documents

- `privacy-policy/`: Privacy Policy
- `terms-of-service/`: Terms of Service
- `tokushoho/`: Specified Commercial Transactions Act notice

Rules:
- Add Markdown files under each `locale/` directory.
- Use `YYYY-MM-DD.md` filenames.
- The app uses the latest file in each `folder/locale/` directory by filename sort order.

Examples:
- `terms-of-service/ja/2026-02-12.md`
- `terms-of-service/en/2026-02-12.md`
- `privacy-policy/ja/2026-02-12.md`
- `privacy-policy/en/2026-02-12.md`
- `tokushoho/ja/2026-03-07.md`
